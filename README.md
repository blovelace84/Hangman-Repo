This is my Hangman game in Python.
Hangman is a Python-based word-guessing game where players try to uncover a hidden word by guessing letters within a limited number of attempts. 
The project showcases fundamental programming concepts such as loops, conditionals, and user input handling, making it an excellent educational tool for beginner developers. 
It includes customizable word lists and a simple, interactive terminal interface.

Installation:
1. Clone Repository: git clone https://github.com/blovelace84/Hangman-Repo.git
2. Navigate to repository: cd Hangman-Repo
3. Run Application: python hangman.py
